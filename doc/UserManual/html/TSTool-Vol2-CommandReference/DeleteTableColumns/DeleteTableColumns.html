Summary

The DeleteTableColumns() deletes specified columns from a table.

Command Editor

The following dialog is used to edit the command and illustrates the syntax of the command.


DeleteTableColumns() Command Editor

Command Parameters

The command syntax is as follows:

DeleteTableColumns(Parameter=Value,...)

Parameter

Description

Default

TableID

The table identifier for the table to process. Can specify with ${Property}.

None - must be specified.

DeleteColumns

The names of columns to delete, separated by commas. Can use ${Property} to specify column names.

None - must be specified.

Examples

A simple comma-separated-value data file is as follows and can be read with ReadTableFromDelimitedFile(): 


# Simple table for testing
"string1","double1","integer1"
"String1",1.0,1
"String2",2.0,2
"String3",3.0,3

The command file to read the above file remove the last two columns is as follows: 


ReadTableFromDelimitedFile(TableID="Table1",InputFile="testtable.csv")
DeleteTableColumns(TableID="Table1",DeleteColumns="double1,integer1")
