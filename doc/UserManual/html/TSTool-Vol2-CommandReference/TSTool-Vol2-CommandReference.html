TSTool Command Reference

Version 11.10.00, 2016-03-30

Contents

The following lists TSTool commands in alphabetical order. HTML documentation is being prototyped. Refer to the PDF documentation for complete command documentation. In the future documentation may be provided grouped by functionality.

  • ReadReclamationHDB - read time series and ensembles from Reclamation HDB datastore
  • SortTimeSeries - sort time series by identifier or properties
  • WriteReclamationHDB - write time series and ensembles to Reclamation HDB datastore
